void PulseCaptureConfig ( )

Function that configures Timer 0 and D4 GPIO pin as pin change interrupt.

Timer 0 is free running and is reseted only by readPulseCount(). Function readPulseCount() is called in ISR( PCINT2_vect ), which reacts to pin change event.

◆ g_TIM0_ov

uint16_t g_TIM0_ov = 0

Timer 0 overflow counter global variable.

Variable is used to increase limited timer resolution to more then 8 bits. Value is incremented by ISR( TIMER0_COMPA_vect ). Variable is read and written to 0 by readPulseCount() function.
